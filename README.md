## 项目使用技术

- **开发语言**：Swift
- **开发框架**：SwiftUI、AppKit
- **开发工具**：Xcode
- **运行平台**：macOS

## 项目主要功能

TinyDict 是一个轻量级的 macOS 菜单栏词典应用，主要用于快速查询选中的单词或文本。

### 具体功能包括：

- 在 macOS 顶部菜单栏常驻显示应用图标
- 点击菜单栏图标可显示或隐藏词典窗口
- 提供独立的浮动查询窗口，方便随时查看查询结果
- 支持全局快捷键 **Command + Option + S** 进行快速查词
- 快捷键触发后，可自动复制当前选中的文本并读取剪贴板内容
- 调用词典查询模块，对选中的单词进行查询
- 支持通过 macOS 系统的 **Services（服务）** 功能进行查词
- 关闭窗口时不会退出应用，而是隐藏窗口，方便再次快速打开

## 项目特点

- 体积轻量，操作方便
- 适合作为桌面端快速查词工具
- 结合菜单栏、快捷键和系统服务，提高查词效率
- 使用 SwiftUI 构建界面，并结合 AppKit 实现 macOS 原生功能
